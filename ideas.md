# GreenFit Landing Page - Design Brainstorming

## Resposta 1: Biofilia Moderna com Neomorfismo Suave (Probabilidade: 0.08)

**Design Movement:** Biophilic Design + Soft Neomorphism
- Conexão profunda com a natureza através de formas orgânicas, cores naturais e elementos vivos
- Neomorfismo suave para modernidade sem frieza

**Core Principles:**
1. Naturalidade Integrada: Formas fluidas, curvas suaves, elementos que remetem à natureza
2. Acessibilidade Emocional: Acolhimento, confiança e sensação de comunidade
3. Hierarquia Orgânica: Layouts que fluem naturalmente, sem rigidez de grid
4. Sustentabilidade Visual: Cada elemento tem propósito, nada é descartável

**Color Philosophy:**
- Verde Sálvia (primário): #5A8C5A - calma, confiança, natureza
- Verde Esmeralda (destaque): #2D6A4F - energia, crescimento
- Terracota (#B8714F): humanidade, terra, acolhimento
- Bege/Off-white (#F5F1E8): fundo limpo, respiração visual
- Preto Suave (#1A1A1A): texto, sem dureza

**Layout Paradigm:**
- Seções com diagonal cuts e wave dividers (não linhas retas)
- Assimetria intencional: imagem grande à esquerda, texto à direita (alternando)
- Espaço negativo generoso para respiração
- Elementos flutuantes e sobrepostos para profundidade

**Signature Elements:**
1. Folhas decorativas que aparecem em cantos e bordas
2. Linhas animadas conectando passos (como raízes crescendo)
3. Árvores SVG pequenas em diferentes estados de crescimento

**Interaction Philosophy:**
- Scroll-triggered animations: elementos crescem/aparecem conforme usuário desce
- Hover suave: cards ganham sombra neomórfica, cores mudam gradualmente
- Feedback tátil: botões respondem com micro-movimento

**Animation:**
- Fade-in com slide-up ao entrar na viewport (duração: 0.6s)
- Árvore crescendo no hero (raízes e galhos animados)
- Contadores que incrementam quando seção fica visível
- Folhas caindo sutilmente no fundo (parallax leve)
- Transições suaves entre cores (não abruptas)

**Typography System:**
- Display: Fraunces Bold (títulos grandes, impactantes)
- Headlines: Manrope SemiBold (seções, h2/h3)
- Body: Inter Regular (texto corrido, legibilidade)
- Números/Destaques: Caveat (para números de impacto, árvores plantadas)

## Resposta 2: Minimalismo Sustentável com Tipografia Forte (Probabilidade: 0.07)

**Design Movement:** Sustainable Minimalism + Bold Typography
- Menos é mais, mas cada elemento comunica força
- Tipografia como protagonista

**Core Principles:**
1. Clareza Radical: Apenas o essencial, sem decoração supérflua
2. Força Tipográfica: Letras grandes, pesos variados, impacto visual
3. Espaço Generoso: Muita respiração entre elementos
4. Contraste Inteligente: Cores sólidas, sem gradientes

**Color Philosophy:**
- Verde Escuro (#1B4332): força, confiança, profundidade
- Branco Puro (#FFFFFF): clareza, transparência
- Tons de Cinza (#E8E8E8, #666666): hierarquia suave
- Destaque em Laranja Terroso (#D4734A): urgência positiva

**Layout Paradigm:**
- Grid 2-coluna em desktop, 1-coluna em mobile
- Tipografia como divisor de seções (não linhas)
- Imagens grandes e ousadas, ocupando metade da tela
- Alinhamento à esquerda (não centralizado)

**Signature Elements:**
1. Números enormes em Caveat (árvores plantadas)
2. Linhas horizontais grossas em verde (separadores)
3. Ícones minimalistas (apenas contorno)

**Interaction Philosophy:**
- Hover: mudança de cor sólida (não gradual)
- Cliques: feedback imediato com mudança de escala
- Scroll: elementos aparecem com fade simples

**Animation:**
- Fade-in rápido (0.3s) ao entrar na viewport
- Contadores com easing linear (não suave)
- Sem parallax, sem excesso de movimento

**Typography System:**
- Display: Fraunces Black (títulos, peso máximo)
- Headlines: Manrope Bold (h2/h3, peso forte)
- Body: Inter Medium (texto, peso elevado para legibilidade)
- Números: Caveat Bold (impacto máximo)

## Resposta 3: Playful Organic com Gradientes Suaves (Probabilidade: 0.06)

**Design Movement:** Playful Organicism + Soft Gradients
- Diversão e energia sem perder a seriedade do propósito
- Gradientes sutis que remetem à natureza

**Core Principles:**
1. Energia Positiva: Cores vibrantes mas naturais, movimento constante
2. Formas Lúdicas: Bolhas, ondas, formas arredondadas extremas
3. Comunidade Visual: Personagens, ilustrações, humanização
4. Movimento Contínuo: Animações suaves em loop

**Color Philosophy:**
- Verde Lima (#A8D5BA): energia, frescor, juventude
- Verde Água (#6FCCCC): tranquilidade, fluxo
- Amarelo Suave (#F4D9A6): otimismo, calor
- Roxo Pastel (#D4A5D4): criatividade, diferença
- Fundo: Gradiente suave de verde água para bege

**Layout Paradigm:**
- Seções com shapes orgânicas (não retângulos)
- Elementos flutuantes em diferentes profundidades
- Sobreposição intencional de cards e imagens
- Movimento de parallax em múltiplas camadas

**Signature Elements:**
1. Bolhas decorativas em diferentes tamanhos
2. Ondas animadas como divisores de seção
3. Personagens ilustrados (avatares dos professores)

**Interaction Philosophy:**
- Hover: elementos "flutuam" para cima com sombra
- Cliques: feedback com bounce animation
- Scroll: elementos giram, escalam, mudam cor

**Animation:**
- Bounce ao entrar na viewport (0.8s)
- Elementos em loop contínuo (respiração, rotação suave)
- Parallax agressivo em imagens de fundo
- Transições com easing bounce

**Typography System:**
- Display: Plus Jakarta Sans Bold (títulos, modernidade)
- Headlines: Manrope SemiBold (h2/h3)
- Body: Inter Regular (texto, legibilidade)
- Destaques: Caveat (números, interações)

---

## Design Escolhido: **Biofilia Moderna com Neomorfismo Suave** ✅

Escolhi a **Resposta 1** porque:
- Alinha perfeitamente com o propósito da marca (natureza + sustentabilidade)
- Cria conexão emocional sem ser infantil
- Oferece sofisticação moderna (neomorfismo) com acolhimento (biofilia)
- Permite microinterações naturais e fluidas
- Diferencia-se de landing pages genéricas

### Decisões Implementadas:
- **Cores:** Verde Sálvia primário, Esmeralda para CTAs, Terracota para detalhes, Off-white para fundo
- **Tipografia:** Fraunces para títulos (impacto), Manrope para seções (modernidade), Inter para corpo (legibilidade)
- **Formas:** Cantos arredondados generosos, wave dividers, folhas decorativas
- **Animações:** Scroll-triggered, fade-in/up, contadores, árvore crescendo
- **Layout:** Assimétrico, espaço generoso, elementos flutuantes
